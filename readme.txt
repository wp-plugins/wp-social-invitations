== Description ==

Allow your visitors to invite friends of their social networks such as  Google, Yahoo, Hotmail and more directly into your Wordpress site. This plugin works perfectly with Buddypress.

Features

*Widget with shortcode or php function
*Defaults invitations messages and the ability to disable user edition
*Custom CSS
*Translation ready
*Documentation

Check premium version on [http://www.timersys.com/wordpress-plugins/wordpress-social-invitations/](http://www.timersys.com/wordpress-plugins/wordpress-social-invitations/)

== Installation ==

1. Upload the `wp-social-invitations` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

You're done!
